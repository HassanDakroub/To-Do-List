This is a simple and interactive To-Do List web application built using HTML, CSS, and JavaScript. It allows users to easily manage their daily tasks by adding, editing, and deleting items in a clean and user-friendly interface.

The project focuses on practicing DOM manipulation, event handling, and dynamic UI updates using vanilla JavaScript.

⚙️ Features
➕ Add new tasks
✏ Edit existing tasks
❌ Delete tasks
🖱 Interactive and simple UI
🎯 Lightweight and fast performance
🛠️ Technologies Used
HTML5 → Structure of the app
CSS3 → Styling and layout design
JavaScript (Vanilla JS) → Functionality and DOM manipulation
🎯 Purpose of the Project

The goal of this project is to improve frontend development skills by building a real-world mini application using core web technologies without frameworks.

It helps understand how JavaScript interacts with HTML elements and how dynamic web applications work.

🌟 What I Learned
DOM manipulation
Event handling in JavaScript
Creating dynamic elements
Structuring a small web application
Improving UI/UX design basics
